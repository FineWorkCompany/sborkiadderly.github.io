Сборки Windows 8.1

Windows 8.1 Pro V6 by Adderly (09.10.2022)

Добавлена папка "от Марка Аддерли", в которой находятся инсталяторы полезных и авторских программ, а так же Reg файлы, чтобы редактировать проводник и вернуть старый просмотрщик фото, чтобы отключать и включать обновления.

Интеграции:
Microsoft Visual Studio Redistributable 2005, 2008, 2010, 2012, 2013, 2015 - 2022, x86 и x64
Framework 4.8, 4.7.2, 3.5, 3.0, 2.0

Включен DirectPLay

Добавлены темы "Архитектура, Персонажи, Ландшафт" из Windows 7
Добавлены темы с обоями рабочего стола из Windows Vista, 7, 8.0, 10, 11
Добавлены темы с обоями из экрана блокировки Windows 8 и Windows 10
(по желанию можно удалить)

Добавлен диспетчер задач в ПКМ по рабочему столу
(по желанию можно отключить)

Ключ - GCRJD-8NW9H-F2CDX-CCM8D-9D6T9

Образ весит 5.00 GB, после установки Windows занимает 15.8 GB.

Образ основан на официальном MSDN Windows 8.1 с Update 3.
